# Neural-Network-using-OS-concepts

## Requirements
- C++
### - Libraries used:
* #include <cstdlib.h>
* #include <unistd.h>
* #include <sys/types.h>
* #include <iostream>
* #include <fstream>
* #include <string.h>
* #include <sstream>
* #include <vector>
* #include <pthread.h>
* #include <fcntl.h>
* #include <sys/stat.h>


## Setup Instructions
* Compile and run the Main.cpp file.
* g++ -o m Main.cpp
* ./m
